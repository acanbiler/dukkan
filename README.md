# Dukkan - Shopping Web Application

A modern shopping web application built with React and Spring Boot microservices.

## Tech Stack

### Frontend
- React 19.2
- Mantine UI 8.3.7
- TypeScript
- React Router v7
- Vite

### Backend
- Java 25
- Spring Boot 3.5.7
- Spring Cloud Gateway
- Maven
- Microservices Architecture

### Data Layer
- PostgreSQL 17
- Liquibase (migrations)

### Infrastructure
- Docker & Docker Compose

## Project Structure

```
dukkan/
├── backend/
│   ├── api-gateway/          # Spring Cloud Gateway
│   └── product-service/       # Product catalog and inventory
├── frontend/                  # React application
└── docker/                    # Docker configurations
```

## Services

- **API Gateway**: Entry point for all backend services
- **Product Service**: Manages product catalog, inventory, and categories
